Maratona em Movimento: Um Site Interativo sobre Corridas

Este projeto apresenta uma página web dedicada ao universo das maratonas de corrida. A plataforma fornece informações sobre eventos nacionais e internacionais, dicas de preparação, nutrição e treinos, além de um simulador interativo para calcular ritmo de prova e tempo estimado de conclusão. Desenvolvido com HTML, CSS e JavaScript.
